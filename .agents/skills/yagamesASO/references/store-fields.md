# Yandex Games Store Fields

Use these fields for each language.

| Field | Russian UI label | Maximum characters | Guidance |
| --- | --- | ---: | --- |
| Title | Название | 50 | Game name only. |
| SEO description | Описание для SEO | 160 | Search-friendly summary of genre and core loop. |
| About the game | Об игре | 1000 | Store description with verified content and features. |
| Short description | Короткое описание | 70 | Compact one-sentence pitch. |
| How to play | Как играть | 1000 | Dry controls only unless the user explicitly asks for rules. |

## Screenshot Requirements

Prepare two different screenshots for each language and each required orientation.

### Landscape

- Ratio: `16:9`
- Recommended export: `1920x1080`
- Accepted long side: `1280-2560 px`
- Format: JPEG or `24-bit PNG` without alpha

### Portrait

- Ratio: `9:16`
- Recommended export: `1080x1920`
- Accepted long side: `1280-2560 px`
- Format: JPEG or `24-bit PNG` without alpha

## Copy Checklist

- Keep RU and EN claims equivalent.
- Verify content counts from source rather than estimating them.
- Exclude disabled or future features.
- Keep `How to play` limited to desktop and mobile input methods.
- Count characters after the final wording is ready.
